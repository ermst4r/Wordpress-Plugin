Installeer een High Quality Affiliate site met Alfieliate

== Description ==
Alfieliate is een tool wat datafeeds verrijkt en datafeeds ontwikkeld met extra attributen. Onder extra attributen wordt verstaan dat producten in datafeeds worden verreken met extra waardes. Neem als voorbeeld een BH, Alfieliate  voegt dan automatisch waardes toe zoals Type BH, kleur, materiaal etc. Op deze manier kunt u een High Quality affiliate website ontwikkelen.

**What is Alfieliate**
Voor Alfieliate hebben wij een Wordpress Plugin beschikbaar. Hiervoor dient u binnen Alfieliate een API sleutel aan te schaffen. D.m.v. de API sleutel wordt er rechtstreeks contact gemaakt met ons systeem, u hoeft dan de filter alleen te installeren op uw website.


De attributen die bij een filter horen worden rechtstreeks getoond op uw website zodat de content indexeerbaar is. Daarnaast staat alle data in uw eigen database. Door middel van de API sleutel kunt u ook gebruikmaken van onze API Calls, u kunt dan d.m.v. JSON eigen requests uitvoeren.
